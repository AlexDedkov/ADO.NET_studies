1. Используется база данных FarmersMarkets SQL Server созданная с помощью farmersmarkets.sql
2. Создать пользователя SQL сервера с данными: Username: Alex Password: privet123 с доступом к базе данных farmersmarkets
3. Запустить приложение с помощью FarmersMarketsApp.exe
4. В данном приложении представлены 2 таблицы: category и market.
4.1 Для переключения между ними используйте окошко выбора над кнопкой Save
4.2 Для изменения данных в таблице дважды кликнете по столбцу и измените значение. ВАЖНО: category_id и market_id устанавливаются автоматически, при попытке редактирования будет ошибка
4.3 Для сохранения изменений нажмите Save. При успешном сохранении появится окошко подтверждения. В другом случае будет показана ошибка.
4.4 Для добавления строк нажмите кнопку Add. При добавлении строки изменения будут сохранены автоматически. Строка будет добавлена снизу таблицы.
4.5 Для удаления строк из таблицы выберете строку и нажмите кнопку Delete. При удалении строки изменения будут сохранены автоматически.
